# 익키 

익숙한 키오스크의 앞글자를 따서 익키라는 이름을 사용하게 되었습니다.
기존의 키오스크 화면은 불필요한 화면전환이 많다고 생각되어 꼭 필요한 화면인 메뉴판만 넣어서 개발하게 되었습니다.
